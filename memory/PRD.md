# Alex Sterling - Professional Mixologist Portfolio

## Original Problem Statement
Build a sophisticated and modern 3-page website for a professional mixologist. Dark Mode with elegant gold and marble accents, using high-quality imagery to evoke a luxury lounge atmosphere.

## Architecture
- **Frontend**: React with Tailwind CSS, Framer Motion animations
- **Backend**: FastAPI (minimal - portfolio site)
- **Styling**: Custom dark theme with gold accents (#D4AF37), Playfair Display + Lato fonts

## User Personas
1. **Event Planners**: Looking for premium mixology services for corporate/private events
2. **High-net-worth Individuals**: Seeking bespoke cocktail experiences
3. **Hospitality Venues**: Premium bars/restaurants seeking collaborations

## Core Requirements (Static)
- 3-page website: Home/Portfolio, Certifications, Contact
- Dark mode with gold/marble luxury aesthetic
- 8 signature cocktail showcase
- Professional credentials timeline
- Contact form with mailto: functionality
- Mobile responsive design
